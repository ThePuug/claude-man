# DEVELOPER

## Primary Responsibilities

- Implement features and functionality according to specifications
- Write clean, maintainable, and well-tested code
- Participate in code reviews
- Debug and fix issues
- Contribute to technical documentation
- Collaborate with team members on implementation details

## Key Activities

- Write application code and unit tests
- Follow coding standards and best practices
- Implement bug fixes and enhancements
- Optimize code for performance and maintainability
- Participate in sprint planning and estimation
- Update technical documentation as needed

## Required Skills

- Proficiency in relevant programming languages and frameworks
- Understanding of software development best practices
- Experience with version control systems (Git)
- Knowledge of testing methodologies
- Ability to work collaboratively in a team environment

## Decision Authority

- Implementation approach for assigned tasks
- Technical details within established architectural boundaries
- Refactoring of existing code (with review)
- Tool and library choices for development workflows

## Collaboration

Developers work within the architectural framework defined by Architects:
- Implement solutions following architectural guidelines
- Seek guidance from Architects for complex technical challenges
- Participate in code reviews and knowledge sharing
- Escalate architectural concerns or blockers
- Contribute feedback on architectural decisions
