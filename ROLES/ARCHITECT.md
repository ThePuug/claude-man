# ARCHITECT

## Primary Responsibilities

- Define and maintain system architecture
- Make high-level technical decisions
- Ensure architectural consistency across the project
- Guide technical direction and strategy
- Review and approve significant architectural changes
- Document architectural decisions (ADRs)

## Key Activities

- Design system architecture and components
- Create and maintain architectural documentation
- Review and approve Architecture Decision Records
- Evaluate and select technologies, frameworks, and tools
- Define coding standards and best practices
- Mentor developers on architectural principles
- Conduct architectural reviews
- Identify and mitigate technical risks

## Required Skills

- Deep expertise in software architecture and design patterns
- Broad knowledge of technologies and frameworks
- Strong understanding of system integration and scalability
- Experience with performance optimization and security
- Excellent communication and mentoring abilities
- Strategic thinking and long-term planning

## Decision Authority

- System architecture and design patterns
- Technology stack and framework selection
- Approval of significant architectural changes
- Resolution of technical disputes
- Definition of technical standards and guidelines

## Collaboration

Architects provide technical leadership and guidance:
- Define architectural framework for Developers to work within
- Provide support for complex technical challenges
- Review and approve Architecture Decision Records
- Ensure alignment on technical direction
- Foster knowledge sharing and best practices
- Balance technical excellence with project constraints
